- Crossmatch с TIC каталогом → TIC 610495795
- Проверка через TESScut: separation 1.8 пикселя = та же звезда

### 3. Очистка данных
- Убрано лунное окно: BTJD 3215-3221
- Обрезаны края: +0.5d начало, -1d конец
- 3-sigma clipping
- Итого: 1135 точек из ~1700

### 4. Графики
- Raw lightcurve: показаны затмения, отмечено лунное окно
- Phase-folded: 0-2 цикла, 200 бинов, видны primary + secondary

### 5. Советы от друга (применены)

- Robust range через перцентили (5-95%)

- Legend снаружи графика

## Уроки

1. **Проверяй в VSX сначала** — звезда может быть уже известна
2. **Используй magnitude** — стандарт для VSX
3. **Fine binning** — 50 бинов мало, 200 лучше для затмений
4. **Робастный range** — перцентили, не min/max
5. **Не overclaim тип** — лучше EA: чем EA если не 100% уверен
6. **Epoch на минимуме** — для EA всегда primary eclipse
7. **Указывай BJD/HJD** — честно в remarks
